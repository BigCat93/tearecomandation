# 个性化茶饮推荐系统

这是一个基于Python的个性化茶饮推荐系统，结合中医九种体质理论，为用户提供个性化的茶饮推荐。

## 功能特点

- 中医九种体质知识库
- 个性化茶饮推荐
- 基于决策树和随机森林的分类算法
- 响应式Web界面
- 跨平台兼容

## 安装说明

1. 克隆仓库：
```bash
git clone [您的仓库URL]
cd tea_recommender
```

2. 创建虚拟环境：
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 运行应用：
```bash
python app.py
```

## 系统要求

- Python 3.8+
- 支持现代浏览器

## 使用说明

1. 访问首页，了解中医九种体质基本知识
2. 点击"今天喝什么茶？"开始个性化推荐
3. 填写个人健康数据（约需1分钟）
4. 获取个性化茶饮推荐结果
